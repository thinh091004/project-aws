* Grasped core EC2 architecture components including AMI selection, Instance types, and EBS block storage.

* Set up and proficiently utilized AWS Cloud9 as a primary cloud-hosted development environment.

* Acquired practical skills in integrating and orchestrating secure resource communication between compute (EC2) and storage (S3) services.
