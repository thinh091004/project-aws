Below is my complete Worklog throughout the internship program as part of the First Cloud AI Journey (FCAJ) project.

How was it completed? (How): The program was carried out following a progressive hands-on approach: starting from mastering core AWS cloud infrastructure, security & performance optimization, to engineering Serverless architectures, integrating Full-stack systems (Frontend & API Gateway), deploying an automated Scoring Engine, and automating End-to-End testing with Postman.

Duration: 12 weeks.

Weekly Breakdown:

**Week 1:** [Getting Familiar with AWS, Account Management & Amazon VPC Network Design](1.1-week1/)

**Week 2:** [Amazon EC2 Virtual Servers, IAM Roles Authorization, AWS Cloud9 & S3 Static Web Hosting](1.2-week2/)

**Week 3:** [Amazon RDS, DynamoDB Databases & Deployments with Amazon Lightsail](1.3-week3/)

**Week 4:** [Automated Scaling with EC2 Auto Scaling, CloudWatch Monitoring & AWS CLI Management](1.4-week4/)

**Week 5:** [Performance Optimization via ElastiCache, CloudFront CDN Acceleration & Networking Workshop](1.5-week5/)

**Week 6:** [Enterprise Services (Windows/AD), Data Migration (DMS/SCT) & High Availability (HA) Architecture](1.6-week6/)

**Week 7:** [API Gateway Security with Amazon Cognito, Frontend Auth Integration & Postman Smoke Tests](1.7-week7/)

**Week 8:** [Header Authorization Token Propagation, Frontend API Connectivity & Scoring Lambda Engine](1.8-week8/)

**Week 9:** [Cognito Groups Authorization (RBAC), Scoring Dashboard & Postman Automated Workflow Tests](1.9-week9/)

**Week 10:** [API Security Hardening (CORS/Throttling), E2E Testing via Postman Runner & UX Polish](1.10-week10/)

**Week 11:** [System Code Freeze, AWS Resource Cleanup, Reporting & Live Demo Project Sign-off](1.11-week11/)

**Week 12:** [Post-deployment monitoring, infrastructure maintenance, performance evaluation & lessons learned](1.12-week12/)
